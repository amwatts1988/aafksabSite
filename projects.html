By Jhon Doe

 

I will be going to CanSecWest in March to participate in Pwn2Own 2019. My main focus will be the Tesla Model 3 challenge. I will also be bringing something for one of the Microsoft challenges as well.

My partner will be my Brother in law Adam, he is a developer and specializes in the .Net Framework in a production environment.

 

I plan to build an Arduino Can-Bus interface and a harness to connect into the Can-Bus network on the Model 3. I needed to get up to speed on Arduino and one of my buddies gave me an Uno R3. Now I needed some parts to tinker with and the Can-Bus shield from ElecFreaks.

 

I ordered a “Sun Founder lcd106 starter kit” and needed to complete the “Hello, World!” project first.

I ended up completing 8 of the projects from this kit to gain a solid base of info and debugging errors.

I had to change a few of the pins around because my hardware was slightly different, but the end result was the same for each project. After I thought I had got a feel for Arduino I decided to extend the sketch with a scrolling message and a clock based off of a sub routine that counts milliseconds. I will be storing this sketch on my GitHub.

 

While my iron was still hot from the Sun Founder kit I also finished soldering the header pins on the Can-Bus board. I did have to redo a few her and there where I added a little too much solder.. lol I really like the solder sucker tool and would highly recommend one if, like me, you get a little heavy handed with the solder or tin. The board was purchased from ElecFreaks.

 

For this project I decided to try ChipQuick .8mm no clean flux core solder.

I have worked with some small electronics in the past so this was not too difficult of a task to begin with. I will note that the ChipQuick solder was pretty effortless to use in this application. I will be building a custom Cable to connect to the Can-Bus network via a non standard access port, but instead will use wire crimp ends with a little bit of silver solder instead. More to come on this project after I have completed the interface programming.

To help me in my goal of Pwning a Tesla I have purchased “The Car Hackers Handbook” in paper back. This seems to be the most comprehensive manual on the subject. I do have a PDF version of this as well. But some books just need to be physical.

Now, on to programming the interface for use with the python module. I will start with this Instructables guide and work from there. I have a desired goal of using the built in linux driver with Python module to create some custom scripts and tools to interact with the ECM and other systems. I plan to build a logger, tuner, fuzzer, and payload injector.

This is my first time programming a micro-controller myself. However I have built projects in the past from existing projects usually with a few mods, so this should be a fun and challenging adventure.
