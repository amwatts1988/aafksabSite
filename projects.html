I will be going to CanSecWest in March to participate in Pwn2Own 2019. My main focus will be the Tesla Model 3 challenge.

Also I'll be bringing something for one of the Microsoft challenges, if I cana get it working.

My partner will be my Brother in law Adam, he is a developer and specializes in the .Net Framework in a production environment.


I plan to build an Arduino Can-Bus interface and a harness to connect into the Can-Bus network on the Model 3.

Than I needed to get up to speed on Arduino and one of my buddies gave me an Uno R3. From Amazon I ordered some parts to tinker with,

and the Can-Bus shield from ElecFreaks.


My order included a “Sun Founder lcd106 starter kit” so I started the “Hello, World!” project first.

I ended up completing 8 of the projects from this kit to gain a solid base of info and debugging errors.

One snag I ran into was a change to a few of the pins because my hardware was slightly different

The end result was the same for each project. After I thought I had got a feel for Arduino I decided

to extend the sketch with a scrolling message and a clock based off of a sub routine that counts milliseconds.

I will be storing this sketch on my GitHub linked below.


While my iron was still hot from the Sun Founder kit I also finished soldering the header pins on the Can-Bus board.

I had to redo a few her and there because I added a little too much solder.. lol


I really like the solder sucker tool

and would highly recommend one if, like me, you get a little heavy handed with the solder or tin.

The board was purchased from ElecFreaks.


For this project I decided to try ChipQuick .8mm no clean flux core solder.

I have worked with some small electronics in the past so this was not too difficult of a task to begin with.

Note: The ChipQuick solder was pretty effortless to use in this application. I will be building a custom Cable

to connect to the Can-Bus network via a non standard access port, but instead will use wire crimp ends with a

little bit of silver solder instead. More to come on this project after I have completed the interface programming.


To help me in my goal of Pwning a Tesla I have purchased “The Car Hackers Handbook” in paper back. Definitely worth the read, and if you have never checked out No Starch Press I highly suggest you do so!


Now, on to programming the interface for use with the python module. I will start with this Instructables guide

and work from there. I have a desired goal of using the built in linux driver with Python module to create some

custom scripts and tools to interact with the ECM and other systems. I plan to build a logger, tuner, fuzzer,

and finally, a payload injector.


This is my first time programming a micro-controller solo. However I have built projects in the past from existing

projects usually with a few mods, so this should be a fun and challenging adventure.


I will try to complete a post a week but I wouldnt be too suprised if I get a little behind.

Until next time, and have a great week!
